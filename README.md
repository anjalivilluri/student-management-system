# 🎓 Student Management System

A web-based **Student Management System** developed using **Python Flask, HTML, CSS, and SQLite/MySQL**.  
This project helps manage student records efficiently through a simple and user-friendly interface.

---

# 📌 Features

- 🔐 User Login Authentication
- 🏠 Dashboard Page
- ➕ Add Student Details
- 📋 View Student Records
- ✏️ Edit Student Information
- ❌ Delete Student Records
- 📊 Reports Page
- 🎨 Responsive User Interface

---

# 🛠️ Technologies Used

| Technology | Purpose |
|------------|----------|
| Python | Backend Programming |
| Flask | Web Framework |
| HTML | Page Structure |
| CSS | Styling |
| SQLite / MySQL | Database |
| Jinja2 | Template Rendering |

# 🚀 Installation Steps

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/student-management-system.git
```

---

## 2️⃣ Open Project Folder

```bash
cd student-management-system
```

---

## 3️⃣ Install Flask

```bash
pip install flask
```

---

## 4️⃣ Run the Application

```bash
python app.py
```

---

## 5️⃣ Open in Browser

```bash
http://127.0.0.1:5000
```
